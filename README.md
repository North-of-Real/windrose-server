# Windrose Server (Private Project)

I've built a Smalland server and enjoyed having the ability to adjust it as needed. I'm still learning to create Docker images for my self hosting hobby on linux (NixOS), and had not yet created an image that required Wine. I plan to have one of these for Dune Awakening as well, so I decided to use an existing Windrose server as a starting point to learn. I have no idea how well-maintained this will be, so for the moment, I recommend using the one that I am learning from instead.
